# Quick start

```bash
# Install dependencies
pnpm install

# Development (hot reload)
pnpm dev

# Production build
pnpm build

# Run production server
pnpm start

# Lint
pnpm lint
```

The dev server runs at `http://localhost:3000`.

## Pages

- `/` — home page (all marketing sections)
- `/headshots` — full gallery (63 images + video testimonials)
- `/thank-you` — post-quote conversion confirmation
